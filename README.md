## Descripción

Este proyecto implementa una clase en Java para gestionar hábitos personales.
Permite crear hábitos, marcar si están completados y modificar su duración.

## Archivos

* Habitos.java → Clase que modela un hábito.
* ProyectoRutinaHabitos.java → Clase principal para probar el funcionamiento.
